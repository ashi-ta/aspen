ASPEN documentation
===================

ASPEN is a full python toolkit to generate ``Auditory Stimulus for Psychophysical ExperimeNt``.

This project aims to create any auditory signal to examine human perception for research and mainly focuses on stimulus generation itself, not a platform for psychophysical experiment.

.. note::
   
   recommend to be used in conjunction with another excellent package such as `Psychopy <https://www.psychopy.org/>`_ if run psychophysical experiment
